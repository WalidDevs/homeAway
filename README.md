# AirBNB_Project

## Installation des dépendances

Avant tout, assurez-vous d’avoir **Bun** installé :  

### Étapes d’installation :

```bash
# Cloner le projet
git clone https://github.com/miage-amiens-organization/2024_M1_PRO-05_GR10.git

# Se déplacer dans le dossier frontend
cd FrontEnd/myapp

# Installer les dépendances du projet
bun install

# Ajouter les dépendances spécifiques au projet
bun add date-fns
bun add @radix-ui/react-slot
bun add @radix-ui/react-popover
bun add react-calendar
